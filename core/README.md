# 🍿NewSnack
### 최신 핵심 경제 뉴스를 압축적으로, 간편히 제공하는 플랫폼
- 최신 뉴스 핫토픽 -> WordCloud 제공
- 데이터 분석을 통해 기사별 Keyword 추출
- 키워드 Click만으로 query -> 관련 뉴스 탐색
- 기사 요약본 제공
---
### 데이터 분석
- TF-IDF
- FastText
